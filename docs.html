📦 s1024.js Library

The fastest way to integrate Storage1024 into your web apps.

<script src="https://storage1024.onrender.com/s1024.js"></script>
<script>
    s1024.set_token('YOUR_PRIVATE_OR_PUBLIC_TOKEN');
    
    // Save a Global Variable
    s1024.add_gv('theme', 'dark');

    // Fetch a Global Variable
    const theme = await s1024.get_gv('theme');
    console.log(theme); // 'dark'
</script>

🌐 REST API

Direct HTTP endpoints for any language.

GET /projects/{id}/gv/{alias}
POST /projects/{id}/upload
DELETE /projects/{id}/files/{alias}

🔐 Security Levels

  • Private Tokens (;qi=): Full access. Dashboard login, token generation, deletion.
  • Public Tokens (;gg=): Restricted. Can upload and fetch GVs, but cannot delete or revoke.
