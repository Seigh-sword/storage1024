📦 JS Library (s1024.js)

The easiest way to power your frontend.

<script src="/s1024.js"></script>
<script>
  s1024.set_userID('YOUR_ID');
  s1024.set_token('YOUR_TOKEN');

  // Async/Await support
  const bg = await s1024.get_gv('bg_color');
  document.body.style.background = bg;
</script>

🌐 REST API

Endpoint specification for any language.

POST /api/projects/create Create Account
GET /api/projects/{id}/gv Fetch all Variables
POST /api/projects/{id}/tokens Generate Token
POST /api/projects/{id}/upload Secure File Upload
GET /files/{alias} Public CDN Link
